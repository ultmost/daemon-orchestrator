# What-If Oracle - Scenario Analysis

> Explores consequences and possibilities of specific paths or events.

## Triggers
- "what if...", "what would happen if..."
- "scenarios", "scenario analysis"
- "what could go wrong", "best case / worst case"
- "contingency", "risk analysis", "consequences"
- "impact of", "projection"

## Does NOT trigger on
- Choosing between options (that's Council)
- Asking for recommendations (that's Council)

## Process
1. Define the scenario clearly
2. Map branching paths (decision tree)
3. For each branch: probability, impact, timeline
4. Identify best case, worst case, most likely
5. Surface hidden risks and second-order effects
6. Present contingency recommendations

## Output Format
- Scenario definition
- Branch map (visual if possible)
- Per-branch analysis: probability, impact, timeline
- Risk matrix
- Recommended contingencies

## Critical Rules
- Be honest about uncertainty
- Don't collapse to a single prediction
- Consider second and third-order effects
- Flag assumptions explicitly
- Quantify when possible, qualify when not
